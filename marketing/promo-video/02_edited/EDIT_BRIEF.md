## 🎨 ビジュアル仕様

### ブランドカラー (copy.md より)
- **アクセント (字幕・ロゴ強調)**: `#FF6B35` (アラート赤、緊張感)
- **背景グラデ**: `#0F0F1E` → `#1A1A2E` → `#0F3460`
- **字幕本体**: 白 (`#FFFFFF`) ベース、強調語のみ `#FF6B35`

### フォント
- **メイン**: Noto Sans JP Black (太字、緊張感)
- **代替**: ヒラギノ角ゴ W8、または system font の `.bold` ウェイト
- **サイズ感**: 字幕は画面下 1/4 に配置、フォントサイズ 80–100pt (1080×1920 換算)

### 字幕配置
- **位置**: 画面下から 200px の位置、中央寄せ
- **背景**: 半透明黒のバー (`rgba(0,0,0,0.5)`)、字幕の上下に 40px パディング
- **アニメーション**: フェードイン 0.3 秒 → 静止 → フェードアウト 0.3 秒
- **操作 UI を隠さない**: 9 マスグリッドとボタンは見える位置

### トランジション
- **low → mid**: クロスフェード 0.5 秒
- **mid → high**: クイックカット (緊迫感を出すため切り替えを早く)
- **high → ロゴ**: ホワイトフラッシュ 0.2 秒 → ロゴ表示
