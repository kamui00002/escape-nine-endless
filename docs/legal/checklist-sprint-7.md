## 5. 国別法律対応

### 5.1 日本

- [ ] 個人情報保護法 (改正法 2022 年): プライバシーポリシーで対応済
- [ ] 特定商取引法: 個人事業主名 / 屋号 / 連絡先メールを公式サイトに明記
- [ ] 資金決済法: 仮想通貨・ポイント発行なしのため対象外
- [ ] 景品表示法: 「過剰な広告」「優良誤認」に注意。ガチャはないが、シーズンパスの表記には注意

### 5.2 米国

- [ ] COPPA: 4+ レーティングなら厳格適用、13 歳未満からは個人データ収集不可
- [ ] CCPA / CPRA (加州): プライバシーポリシーに「Do Not Sell」セクション、適用閾値未満でも任意で対応
- [ ] CDA Section 230: UGC なしのため大きな影響なし
- [ ] 各州データ侵害通知法: 漏洩時 30-90 日以内の通知義務 (州により異なる)

### 5.3 韓国 (シャットダウン制度緩和傾向)

- [ ] 未成年プレイヤーの深夜時間帯制限 (**2022 年廃止済**、ただし任意で導入推奨)
- [ ] 個人情報保護法: ハングル翻訳必須、Sprint 5 のローカライズと連動
- [ ] ゲーム産業振興法: 韓国国内事業者は等級分類審査必須、海外個人開発者は App Store 経由なら審査済扱い
- [ ] 確率型アイテム表示義務 (2024 年施行): EscapeNine はガチャなしのため対象外 ✅

### 5.4 中国 (版号必須)

- [ ] **個人開発者は版号取得不可** = 国内 App Store / Google Play 配信不可
- [ ] **代替**: TapTap で英語版 / 日本語版ベータ配信 (会議録第 21 回 Marcus 発言)
- [ ] **未成年保護システム** (実名認証、深夜制限): 中国本土配信時のみ必要、当面対応不要
- [ ] 香港・台湾は別ルート (5.5, 5.6 参照)

### 5.5 台湾

- [ ] 個人資料保護法: プライバシーポリシーに繁体字版を用意
- [ ] App Store 経由配信のため、台湾固有の追加審査は不要
- [ ] レーティング: GASP (台湾ゲーム評価) は任意

### 5.6 香港

- [ ] 個人資料 (私隱) 條例: プライバシーポリシーに英語版があれば実質対応可
- [ ] App Store 経由配信のため特別な手続きなし

### 5.7 EU (GDPR)

- [ ] **Consent Mode V2** (Firebase Analytics で既に対応済 ✅)
- [ ] データ主体の権利対応 (アクセス / 削除 / 移転 / 訂正)
- [ ] 14 ヶ月以内のデータ保持 (Firebase デフォルト)
- [ ] **DPA** (データ処理契約): Firebase / AdMob / Meta CAPI とは契約自動締結 (Google / Meta の利用規約に組み込み済)
- [ ] DSA (デジタルサービス法): UGC なしのため軽微対応
- [ ] **オンデバイス ML 戦略のメリット**: 生データを EU 外に送らないため GDPR リスク大幅減 (会議録第 19 回連動)

### 9.2 ドメさんへの推奨

**個人作成 + AI 校正で ¥45,000-60,000 ルート**

理由:

1. 本業 (消防士) との両立で、法務に時間を割きすぎるとコア開発が遅れる
2. EscapeNine は **BtoC 個人開発・買い切り課金中心** で訴訟リスクが極めて低い
3. オンデバイス ML 戦略で GDPR リスク大幅減 (会議録第 19 回)
4. 拒絶通知 / トラブル発生時に**ピンポイントで弁護士・弁理士に相談**するハイブリッド型が最適
5. Sprint 12 以降、海外売上が伸びてから国際出願 + 法務体制強化を検討

### 9.3 弁護士 / 弁理士相談を発動する条件

以下のいずれかが発生したら**ためらわず**有償相談する:

- 商標出願に拒絶理由通知が来た
- 模倣アプリが App Store に出現した
- 個人情報漏洩が発生した
- 海外ユーザーから法的クレームが届いた
- App Store がリジェクト (法務関連の理由) を出した
- メディア取材 / 大手パブリッシャーからオファーが来た
